# Contacts Project

This is the repo to to create , search and delete contacts 

## Project Setup

- Clone the project - `git clone https://github.com/srujanareddyE/react-contacts-app.git`
- Install the dependencies - `npm install`

## Backend Server

The backend server for this in-class project can be found [here](https://github.com/srujanareddyE/react-contacts-server).
